Mission 4

1. Introduction

Cette dernière mission consiste à la réalisation du contre rendu de notre SAE en base de données, avec ses 3 missions. La particularité de ce rapport est qu’il est sous forme de document web, là où chaque page correspond à une partie : introduction, synthèse des différentes missions, conclusion… Pour la réalisation de ce rapport, il a fallu s’approprier les langages HTML pour la rédaction et CSS pour le style, afin de concevoir un site web à la fois pratique, élégant et agréable.

2. Les images

Pour la réalisation de ce site, nous avons inséré de nombreuses images. Les images ont été ajoutées dans le code de la page HTML, et le style (taille, couleur, position) dans la page CSS. Cette manipulation est possible car nous avons préalablement enregistré les images dans le dossier 'images' situé dans le répertoire 'assets' de WebStorm.

  • Dans la page d'accueil, l'image en background est une image assez neutre dans avec des coleurs dans le thème du site. Elle est située en haut de la page, et introduit le titre. Elle permet d'apporter de l'esthétisme, et donne du relief à la page.
  • Dans la partie "A PROPOS", nous avons ajouté des images de personnages pour nous représenter et rendre le site plus immersif. Les images sont positionnées de façon alterné grace à la « variable » CSS grid-area.
    Nous avons ajouté un effet qui apparait lorsque la souris passe au dessus des images, rendant ainsi le site plus attractif. L'image qui est en noir et blanc retrouve ces couleurs après le passage de la souris.
